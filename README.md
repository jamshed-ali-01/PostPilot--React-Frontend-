# PostFlow AI - Social Media Automation

AI-powered social media automation platform for local businesses. Upload job photos and testimonials, then automatically generate and schedule geo-targeted social media posts.

## Features

- Business Admin Dashboard with insights and analytics
- Staff Dashboard for easy content submission
- AI-generated captions and hashtags
- Multi-platform scheduling (Facebook, Instagram, LinkedIn)
- Geo-targeted posting by postcode
- Team management and approval workflows

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- Shadcn/ui components

## Getting Started

```bash
npm install
npm run dev
```
